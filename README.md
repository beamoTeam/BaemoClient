# BeamoClient

| **태그**    | **설명**                                             |
| ----------- | ---------------------------------------------------- |
| **feat**  | 기능 추가                                     |
| **fix** | 버그 수정                                     |
| **style** | 코드 포맷, 세미 콜론, 코드 수정이 없는 경우   |
| **design**    | CSS, UI 수정                                     |
| **refactor** | 리팩토링                                            |
| **doc**  | README.md 추가                                     |
| **deps** | dependency 변경                                          |
| **chore**  | 빌드 스크립트 추가                                             |
| **rename**    | 파일,폴더명 수정 or 옮기는 작업                                    |
| **remove**    | 파일 삭제                                     |
